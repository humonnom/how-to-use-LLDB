# How to use lldb?   
	Language : C    
	Tool : MAC    
---------------------------------------------------   
## 💻 lldb & gdb   
	unix 기반 시스템: gdb   
	OS X, 리눅스, 윈도우 등: lldb   
   
---------------------------------------------------   
## how to start debug?  
   gcc -g [target file]   
   	ex) gcc -g test.c   
   clang -g [target file]   
   	ex) clang -g test.c   
---------------------------------------------------   
## run lldb   
	lldb [exe file]   
		ex) lldb a.out   
	run   
---------------------------------------------------   
## exit lldb   
	^D (ctrl + d)
